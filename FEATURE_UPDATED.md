# ✨ Feature Updates

## 1. Google OAuth Integration 🔐

- **Dual Login System**:
  - **Student Login**: Standard Google Sign-In for all students.
  - **Admin Login**: Restricted Google Sign-In protected by an `authorizedAdmins` whitelist.
- **Security**: Verifies tokens on the backend to prevent unauthorized access.

## 2. AI Career Advisor 🧠

- **Real-time Chat**: Fully interactive chat interface to discuss career paths.
- **Powered by Gemini**: Integrated **Google Gemini 2.0 Flash** API for high-speed, intelligent responses.
- **Capabilities**:
  - Salary estimates for specific roles (e.g., "Salary for React Dev").
  - Skill roadmaps (e.g., "Roadmap for Data Science").
  - Job/Internship recommendations.

## 3. UI/UX Improvements 🎨

- **Dynamic Navigation**:
  - Red **Logout** button for Admins.
  - Gradient **Dashboard** buttons for Students.
- **Error Handling**: Detailed error messages for API issues.

## 4. Visual Refresh ☀️

- **Standardized Light Theme**: Removed dark mode to ensure a consistent, clean, and professional "NoteHub Light" aesthetic across all modules.
- **Clean Interface**: Simplified navigation by removing theme toggles for a focused user experience.
