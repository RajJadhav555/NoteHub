# NoteHub 📘

**Academic Collaboration Platform**

A minimal, feature-rich platform for students to share notes, collaborate, and get career guidance.

## 🚀 Quick Start

1.  **Start Backend**:
    Run `start-services.bat` (Starts Docker containers for Database and Backend).

2.  **Start Frontend**:
    Run `run-notehub.bat` (Starts local Python server).

    _Open [http://localhost:8000](http://localhost:8000) in your browser._

## 🔑 Key Features

- **Google Login**: Secure login for Students and Admins.
- **Career AI**: Integrated AI Chatbot for career guidance (Powered by Google Gemini).
- **Note Sharing**: Upload and download verified academic notes.
